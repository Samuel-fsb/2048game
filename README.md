# Descrição do Projeto

O jogo 2048 é um quebra-cabeça clássico construído com uma grade 4 × 4. O jogador desliza blocos numerados em potências de 2, combinando peças iguais para somá-las, com o objetivo de alcançar o valor máximo de 2048.

# Funcionalidades do Projeto
- *Deslize Total:* Você pode mover os blocos em quatro direções: cima, baixo, esquerda ou direita. Todos os blocos deslizam o máximo possível na direção escolhida.
- *Fusão de Blocos:* Quando dois blocos com o mesmo número colidem, eles se fundem em um único bloco cujo valor é a soma dos dois
- *Geração Aleatória:* A cada movimento válido, um novo bloco (com o valor 2 ou 4) surge em um dos espaços vazios da grade
- *Fim de Jogo:* O jogo termina quando o tabuleiro fica totalmente preenchido e não há mais movimentos possíveis
